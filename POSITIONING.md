## 8. Risk Factors

| Risk | Mitigation |
|------|-----------|
| API costs stack up ($0.05+/sec Runway, $0.04/image Stability) | Remotion replaces Runway for 90% of Reels (free, local render). Use Runway only for cinematic B-roll. |
| Meta API policy changes could break DM automation | Build engagement features as modular — can swap to ManyChat or chatbot layer if needed |
| Predis.ai and Zeely improving fast | SMG's API-first architecture means you upgrade by swapping APIs, not rebuilding. ElevenLabs > Predis TTS. Runway > Zeely video. |
| Harbor adds video features | Harbor's 3-person team can't match 7 specialized APIs. And they'd need to rebuild from scratch — their architecture is text-only. |
| "Too technical" for non-developer creators | Build CLI-first, then wrap in a simple web dashboard later |
